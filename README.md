## Install

  ```shell
  git clone https://github.com/lzjqsdd/WifiAssist.git
  cd WifiAssist`
  sudo ./install.sh
  ```
## 运行
- 终端执行WifiAssist
- 或dash搜索应用WifiAssit，点击图标执行。

## Uninstall
  ```shell
  ./uninstall.sh
  ```
  （建议卸载按照install.sh中的命令对应着逐个删除。）
